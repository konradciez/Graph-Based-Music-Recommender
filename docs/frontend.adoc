== Constants

API_BASE

API client wrapper

Małe API pomocnicze do wywołań endpointów backendu. Wszystkie funkcje
zwracają Promise z parsowanym JSON-em lub null/[] przy błędach.

Get tracks that match all provided tags and optional year range.

renderExplorer()

Renderuje widok eksploratora tagów i inicjalizuje pierwszy (popular)
rząd.

loadPopularTags()

Pobiera najpopularniejsze tagi i tworzy pierwszy rząd przycisków.

toggleTag()

Obsługuje zaznaczanie/odznaczanie taga. Zapisuje informację o tagOrigins
aby tylko instancja taga w źródłowym rzędzie była podświetlona.

updateRelatedTags()

Pobiera powiązane tagi dla aktualnego zestawu state.selectedTags i
tworzy nowy rząd sugestii.

createTagRow(items, rowIndex)

Tworzy nowy rząd przycisków z listą tagów.

highlightSelectedButtons()

Podświetla tylko te przyciski .tag-pill, które odpowiadają zapamiętanym
origins.

resetExplorer()

Resetuje wybór tagów (przywraca tylko pierwszy rząd popularnych tagów).

fetchTracksForTags()

Pobiera utwory dopasowane do state.selectedTags i zakresu lat i
renderuje wyniki.

renderRecommender()

Renderuje widok Recommender i podłącza handlery.

attachRecommenderHandlers()

Przywiązuje event handlery do elementów w widoku Recommender. Obsługuje:
search autocomplete, przyciski +/- przy filtrach oraz przełącznik
Title/Artist.

selectSuggestion(spotify_id)

Wywoływane po wybraniu sugestii z listy. Czyści pole wyszukiwania,
pobiera dane seed-track i żąda rekomendacji.

renderSeedAndResults(seed, recs)

Renderuje seed (wybrany utwór) oraz listę rekomendacji poniżej.
